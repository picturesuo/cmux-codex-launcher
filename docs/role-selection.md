# Role Selection

- `lead`: use for scope control, task decomposition, git state, and integration.
- `create-ui`: use for Landing, Create, options, comments, artifact layout, and UI evidence.
- `brain-evidence`: use for fixtures, Brain source display, memory chips, privacy labels, and import wording.
- `learn-canvas`: use for Learn-this flows, return-to-Create state, visual outline, and Canvas graph polish.
- `export-verify`: use for export prompt content, test/build/typecheck work, and browser smoke verification.

Add project-specific roles in the relevant profile instead of hardcoding them in the launcher.
